📋 Setup Guide — Connect Google Sheets

One-time setup, ~10 minutes. Free Google account only.

1Go to sheets.google.com → create new sheet → name it F&B P&L.
2Click Extensions → Apps Script. Delete everything and paste this script:
3Click Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone (⚠️ must be "Anyone", NOT "Anyone with Google account") → Deploy → authorise.
4Copy the Web app URL → paste into the Connect box on the dashboard.
5Share the Google Sheet with your boss's Gmail — they can view all data live anytime.
